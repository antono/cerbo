# Design Tokens

## Purpose
Define the shared visual tokens used by modal confirmation dialogs.

## Requirements

### Requirement: Modal confirmation surface tokens
Modal confirmation dialogs MUST use the same surface, border, radius, shadow, and spacing tokens as the current quit confirmation dialog.

### Requirement: Modal confirmation action tokens
Modal confirmation dialogs MUST use a consistent primary/ghost action treatment that matches the quit confirmation dialog.

### Requirement: Modal confirmation focus state tokens
Focused or selected actions in modal confirmation dialogs MUST use the same visible focus treatment as the quit confirmation dialog.
